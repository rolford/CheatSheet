CheatSheet: CSS Selectors

Selector Example Description
* * Selects all elements
element p Selects all p elements
.class .footnote Selects all elements with class="footnote"
#id #footnote-12 Selects the element with the id="footnote 12"
[attribute] [target] Selects all elements with a target attribute

Combinations of CSS Selectors

Groups of Selectors

Selectors can be grouped to specify that all are selected, e.g. p, span, div

Combinators

Combinators are selectors that establish a relationship between two or more simple selectors, such as "A is a child of B" or "A is adjacent to B." For more information, see Combinators.
